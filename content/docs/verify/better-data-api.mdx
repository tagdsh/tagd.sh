## Authentication

Request access at [hello@betterdata.co](mailto:hello@betterdata.co).

## Rate limits and SLA

Contact Better Data for current limits and SLA details.

## Blockchain anchoring

Better Data can anchor verification records to Ethereum/Polygon and include `txHash` evidence in responses.

## Access

Current access is private for enterprise and regulated-industry teams.
